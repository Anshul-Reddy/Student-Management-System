#  Student Management System

A desktop-based Student Management System developed using Python, Tkinter, and MySQL for managing student records efficiently.

##  Project Versions

### Version 1 - Console Based

A command-line application built using Python and MySQL for performing CRUD operations on student records.

### Features

* Add Student Records
* View Student Details
* Search Students
* Update Student Information
* Delete Student Records
* MySQL Database Integration

---

### Version 2 - GUI Based (Tkinter)

A desktop application built using Tkinter and MySQL with an interactive graphical user interface.

### Features

* Add Student Records
* View Student Details
* Search Students
* Update Student Information
* Delete Student Records
* Roll Number Validation
* GUI-based Interface using Tkinter
* MySQL Database Integration

## 🛠 Tech Stack

* Python
* Tkinter
* MySQL

## 📌 Project Overview

This project demonstrates the evolution of a Student Management System from a console-based application to a GUI-based desktop application.

The system allows users to perform CRUD operations on student records while storing data in a MySQL database.

## 📚 Concepts Used

* Database Connectivity
* CRUD Operations
* Tkinter GUI Development
* MySQL Queries
* Exception Handling
* Form Validation
